# 《忍影试炼》角色设置

这份文档整理了当前扩展角色池，便于继续做选角、技能拆解、AI 行为和数值配置。

补充说明：

- 第 13、14、15 条是第 3、4、5 条角色的扩展版说明，已合并。
- 当前一共整理出 12 名唯一角色。
- 其中“暗夜献祭者·渊”带有明显回合制表述，后续若接入实时动作模式，建议统一改为秒数冷却。

## 角色池

1. 裂地狂战·岩
- 定位：战士 / 重装前排
- 核心关键词：狂怒、减速、护盾、防御反打

2. 影刃刺客·夜
- 定位：刺客 / 单体爆发
- 核心关键词：背刺、破绽、隐身、斩杀

3. 血契剑士·绯
- 定位：战士 / 吸血续航
- 核心关键词：低血增益、吸血、护盾、契约压制

4. 机关傀儡师·枢
- 定位：刺客 / 召唤控制
- 核心关键词：傀儡、丝线、拉拽、共鸣强化

5. 雷霆战将·霄
- 定位：战士 / 范围控制
- 核心关键词：感电、领域、击飞、雷击

6. 霜华魔导·凛
- 定位：法师 / 减速控场
- 核心关键词：冻结、减速、冰刺、斩杀线

7. 星轨射手·辰
- 定位：射手 / 远程消耗
- 核心关键词：连射、真伤、陷阱、全图贯穿

8. 炎狱术士·烬
- 定位：法师 / 持续灼烧
- 核心关键词：灼烧、引爆、火海、冷却循环

9. 翠羽游侠·翎
- 定位：射手 / 机动风筝
- 核心关键词：穿透箭、翻滚、藤蔓陷阱、旋转射击

10. 雷霆法尊·曜
- 定位：法师 / 连锁爆发
- 核心关键词：闪电链、感电、麻痹、延迟雷暴

11. 暗夜献祭者·渊
- 定位：召唤师 / 持续攻击
- 核心关键词：幽灵、献祭、中毒、魔王召唤

12. 逐火之蛾·希尔
- 定位：召唤师 / 海量召唤
- 核心关键词：蜉蝣铺场、吞噬成长、花蝶联动
